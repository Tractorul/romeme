# 🇷🇴🍲 romeme

C++ but written by a hungry Romanian.

## 🍩 Features

- `gogosi` instead of `int`
- `sarmale <<` instead of `cout <<`
- `ciorba >>` instead of `cin >>`
- built-in chaos 💀
