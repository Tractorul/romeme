# 🇷🇴🍲 romeme

C++ but written by a hungry Romanian.

## 🍩 Features

- `gogosi` instead of `int`
- `sarmale <<` instead of `cout <<`
- `ciorba >>` instead of `cin >>`
- built-in chaos 💀

## 😂 Example

```cpp
#include "romeme.h"

int main() {
    gogosi x;

    sarmale << "Cate gogosi vrei? ";
    ciorba >> x;

    daca(x > 5) {
        sarmale << "Prea multe 😭" << desert;
    }

    return 0;
}
